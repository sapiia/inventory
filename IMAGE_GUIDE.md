## How to Add Images to Your Orders

### Method 1: Use Sample Images (Easiest)
1. Click **"Create New Order"** button
2. In the form, find the **"Item Image URL"** field
3. Click the **"Sample URLs"** button
4. Choose from pre-loaded sample images (Car, Headphones, Watch, Laptop, etc.)
5. The image will automatically preview below

### Method 2: Use Your Own Image URL
1. Find an image online (Google Images, Unsplash, etc.)
2. Right-click the image and select **"Copy Image Address"**
3. Paste the URL into the **"Item Image URL"** field
4. The image will preview automatically

### Method 3: Upload to Image Hosting
If you have images on your computer:
1. Upload to a free image hosting service:
   - [Imgur](https://imgur.com) - Free, no account needed
   - [ImgBB](https://imgbb.com) - Free, simple upload
   - [Cloudinary](https://cloudinary.com) - Free tier available
2. Copy the direct image URL
3. Paste into the **"Item Image URL"** field

## ðŸ’¡ Tips

- **Image Size**: Images are automatically resized to fit (64x64px in table, 192x192px in details)
- **No Image**: If you don't add an image, a default icon will be shown
- **Broken Links**: If the image URL doesn't work, a placeholder icon appears
- **Preview**: Always check the preview before saving to ensure the image loads correctly

## ðŸ”§ Troubleshooting

**Image not showing?**
- Make sure the URL ends with an image extension (.jpg, .png, .gif, .webp)
- Check if the URL is publicly accessible (not behind a login)
- Try copying the image address again
- Use one of the sample images to test

**Image too large/small?**
- Don't worry! The system automatically resizes all images
- For best results, use square images (1:1 ratio)

## ðŸŽ¯ Best Practices

1. **Use descriptive images** that match your item name
2. **Test the URL** by pasting it in a new browser tab first
3. **Use reliable sources** like Unsplash or Imgur for stability
4. **Keep URLs short** when possible for easier management
5. **Update images** if links break over time
